Class ClasseId

Descrição: Esta classe tem o objetivo de disponibilizar atributos comuns entre elementos de uma pagina. O atributo class, id, e algum atributo personalizado.

NOTA: ESTA CLASSE É USADA COMO HERANÇA DAS DEMAIS, NÃO TEM UTILIDADE, SOMENTE SEUS METODOS E ATRIBUTOS SAO HERDADOS PARA SIMPLIFICAR A CRIAÇÃO DE CLASSES MAIS ESPECIFICAS.

  • Atributos
  • Métodos

String: classe -> Uma string que representa a classe de qualquer elemento html.

String: id -> Uma string que representa o id de qualquer elemento.


CONSTRUTOR: ClasseId(String classe, String id)

Constrói um objeto que vai conter representações de classe, id e atributos.

setClasse(String classe)

Define o valor do atributo class.

setId(String id)

Define o valor do atributo id.

setConteudo(String conteudo)

Define o conteudo de um elemento html.

addAtributo(String atributo)

Adiciona atributos ao elemento html em questão.

getClasse()

Recupera o valor da classe

getId()

Recupera o valor do Id

getConteuto()

Recupera o valor do conteudo

getAtributo()

Recupera o valor dos atributos.

Class Tabela

Descrição: Esta classe cria objetos que conseguem representar um elemento de tabela html.

A classe Tabela herda de ClasseId.

  • Atributos
  • Métodos

Array: cabecalho -> Este define o cabecalho de uma tabela html (thead). Um array simples.

Array[Array] : corpo -> Este define o corpo da tabela html (tbody). Varios arrays dentro de um array, cada array será como uma linha da tabela. NOTA: O TAMANHO (array.length) DE CADA ARRAY INTERNO DEVE SER O MESMO DO CABECALHO.

String: classe -> Uma classe para a tabela html.

String: id -> Um id para a tabela html.

String: classeCabecalho -> A classe do cabecalho (thead) da tabela.

Objeto: opcoes -> Opções usadas pelo DataTable no momento de criacao da tabela.


CONSTRUTOR: Tabela(Array cabecalho, Array[Array] corpo, String classe, String id, String classeCabecalho)

Instancia um objeto Tabela com seu cabecalho e corpo.

var t = new Tabela(['NOME','SOBRENOME'],[['MARCOS', 'FELIPE'],['WILSON', 'JARDIM']], 'table table-hover', '', '')

setClasseCabecalho(String classe)

Define a classe do cabecalho da tabela html.

t.setClasseCabecalho('bg-success');

setCorpo(Array[Array] corpo)

Define o corpo de uma tabela html. Ele deve ter a mesma quantidade de elementos em cada array interno e cada array interno deve ter o tamanho igual o do cabecalho.

t.setCorpo([['JOAO', 'BARBOSA']]);

setRodape(Array rodape)

Define o rodape da tabela, este por padrão não é definido no construtor. É esperado um array com o mesmo tamanho (quantidade de itens) do cabecalho.

t.setRodape(['NOME','SOBRENOME'])

converterMonetario(int indexColuna)

Metodo utilizado para converter uma determinada coluna em monetario, por exemplo, se você quiser converter a primeira coluna use o indice 0.

t.converterMonetario(0); // Converter a primeira coluna

converterPercentual(int indexColuna)

Metodo usado para converter uma determinada coluna em monetario, por exemplo, se você quiser converter a primeira coluna, use o indice 0

t.converterPercentual(0); // Converte a primeira coluna em percentual

desenhaDataTable()

Este metodo funciona com o apoio da biblioteca dataTables, ele desenha uma tabela usando a api do dataTables (claro que a tabela já deve existir no html, veja getTabela())

t.desenhaDataTable();

getTabelaDinamica(int colunaChave, Array valoresMonetarios, int campoAserMovido)

Um dos métodos mais complexos utilizados pelo objeto Tabela. Ele tem 3 campos chaves para configuração, cada um deles determina a forma como a tabela dinamica sera desenhada. O primeiro é a colunaChave, ele vai centralizar registros cujo esta coluna seja identica a outras, por exemplo, se o corpo tem esta aparencia, [['MARCOS', 320], ['FELIPE', 230], ['MARCOS', 420]] isto que dizer que a colunaChave zero vai unificar todos os nomes iguais na coluna, se for a coluna zero, então teremos somente um registro de MARCOS e um de FELIPE.

O segundo parametro define colunas como valores monetarios, ou seja, valores que serão somados para definir um total a cada coluna e automaticamente formatados, como o corpo do exemplo acima a coluna 1 pode ser informada dentro de um array ([1]) como coluna que contem valores monetarios a serem totalizados e convertidos.

O terceiro e ultimo parametro, não o menos importante, move um determinado campo para a posição do campo da colunaChave, onde os valores duplicados são removidos. Não compreendeu ? Vou explicar. Vamos supor que temos o seguinte corpo. [['MARCOS','CC',280],['FELIPE','CC',480],['MARCOS','R$',110]]. Foi escolhida como coluna a ser movida a coluna 1, e a coluna a ser convertida a coluna 2, então nos registros MARCOS CC e R$ serão movidos abaixo do nome, assim como FELIPE recebera abaixo o CC, os valores serao somados e o resultado vai dar uma ideia de tabela dinamina sendo que a representacao sera exibida desta forma [['MARCOS','--',R$ 390],['MARCOS','R$',R$ 110],['MARCOS','CC',R$ 280],['FELIPE','--',R$ 480],['FELIPE','CC',R$ 480]]

Um exemplo de execucao do codigo

t.getTabelaDinamica(0,[2],1)

clickTabelaDinamica()

Metodo usado para definir o evento click a tabela dinamica já criada. Este metodo é totalmente dependente de getTabelaDinamica(). Para funcionar é necessário que a tabela dinamica já esteja desenhada na pagina.

t.clickTabelaDinamica()

getTabela()

Recupera uma representação html de uma tabela, com seu cabecalho, corpo e (se houver) seu rodape, suas classes e id. Caso o corpo não esteja de acordo com o tamanho do cabecalho uma mensagem de erro será retornada.

t.getTabela();

removeColuna(int indexColuna)

Remove a coluna da tabela recebendo seu indice, o valor do indice inicia-se por 0, sendo que a primeira coluna tem indice 0.

t.removeColuna(1);

getColuna(int indexColuna)

Método usado para retornar uma copia do corpo da tabela na coluna solicitada.

var colu = t.getColuna(1);

getColunas(Array colunas)

Método usado para retornar uma copia do corpo da tabela nas colunas solicitadas.

O exemplo abaixo retorna tres campos da tabela.

var colu = t.getColunas([1,2,3]);

filtro(String palavraChave)

Metodo usado para retornar cabecalho e corpo de dados filtrados. Recebe uma string para filtro e retorna [[cabecalho],[[corpo],[corpo]]]

var recebeNovaTabela = t.filtro('Marcos');

setOpcoes(Objeto opcoes)

Permite enviar um objeto "{}" de parametros para ser usado pelo datatables (usar antes de chamar o metodo desenhaDataTable()).

t.setOpcoes({'order':true, 'bPaginate':false});

calculaRodape(Array camposNaoCalculaveis, Array valorDosCamposNaoCalculaveis, Array camposMonetarios)

Este metodo faz o calculo da tabela e coloca as informações no rodape. Ele permite que se envie um array com os indices dos campos que nao serao calculaveis, um array contendo o valor destes campos que nao serao calculaveis (este precisa ter o mesmo tamanho do array dos camposNaoCalculaveis) e também um array com os indices das colunas que contem valores monetarios. Já campos com valores de números inteiros são somados de forma correta. Qualquer campos que não esteja listado será somado como se fosse valores inteiros.Isto faz os valores do rodape serem calculados e preechidos de forma correta sem erros.

t.calculaRodape([0,1,2], [' -- ', ' -- ', 'TOTAL'], [3,4]);

Class Grafico

Descrição: Classe usada para instanciar um objeto Grafico que vai representar graficos do google charts

NOTA: PARA FUNCIONAR É NECESSÁRIO QUE O GOOGLE CHARTS ESTEJA PRESENTE NO DOCUMENTO HTML.

A classe Modal herda de ClasseId.

  • Atributos
  • Métodos

Array[Array]: dados -> Um array aninhado com os dados para gerar o grafico

Object: opcoes -> Um objeto de opções com pares {chave:valor}.

String: classe -> A classe da div do grafico

String: id -> O id que indentificar a div do grafico.


CONSTRUTOR Grafico(Array[Array] dados, Object opcoes, String classe, String id)

Inicia um objeto que vai representar um grafico, é importante que o ID do objeto seja informado pois ele é reponsavel por especificar o local exato da criação do grafico. Para funcionar corretamente é necessário que exista uma div com um id, e este ID seja informado na criacao do objeto grafico.

var g = new Grafico([['NOME', 'TOTAL'],['MARCOS',250.00]], {title:'VENDAS DE PRODUTOS'}, '', 'graf');

addColuna(Array dadosColuna);

Ao enviar um array pode-se incluir uma terceira coluna para os dados, casos usados quando a terceira coluna ainda não havia sido criada e ela precisa ser inclusa no grafico. Este array tem que ter o mesmo tamanho da quantidade de arrays dentro do arrayDados.

g.addColuna(['PERCENTUAL', 50.0]);

setCores(Array cores)

Caso queira definir as cores do grafico pode-se enviar as cores desejadas como um array simples e o mesmo será usados para gerar as cores.

g.setCores(['red','green']);

getPizza()

Este metodo cria um grafico de pizza na div com o ID informado no momento da criação do objeto.

g.getPizza();

getBarra()

Este metodo cria um grafico de barra na div com o ID informado no momento da criação do objeto.

g.getBarra()

getColuna()

Este método cria um grafico de coluna na div com o ID informado no momento da criação do objeto.

g.getColuna()

getLinha()

Este método cria um grafico de linha na div com o ID informado no momento da criação do objeto. Este gráfico de linha para ser desenhado corretamente precisa que o arrayDados seja algo constante no eixo X.Ex: Se quero desenhar os rendimentos mensais de uma loja, devo informar que a primeira coluna sera o nome do mes, a segunda no titulo sera o nome da loja(Valor do rotulo) e cada array apos este será a fusao do mes e o valor daquela loja no referente mes. Como uma tabela.

MESLOJA 01
JAN250.00
FEV200.00
MAR650.00
ABR250.00
MAI260.00
JUN450.00

Desta forma da tabela os dados são descritos na forma correta para o exemplo do grafico

g.getLinha();

Class Formulario

Descrição: Classe usada para instanciar um objeto Formulario que vai validar campos de formularios.

A classe Modal herda de ClasseId.

  • Atributos
  • Métodos


CONSTRUTOR Formulario()

Inicia um objeto formulario que é capaz de validar datas e campos. Ele não recebe nenhum parametro.

var f = new Formulario();

validaData(idOuClasse);

Valida a data informada retornando um objeto Date se a data estiver correta, senão retorna false.

var da = f.validaData('data1');

deMenorQueAte(data1, data2)

Verifica se a data1 é menor que a data2. Usado para validar campos de data em filtros de/ate. Os parametros enviados são idOuClasse de cada campo de formulario date. Se tudo ocorreu bem retorna true e atribui ao objeto Formulario os valores de data1 e data2, sendo que cria os atributos de = data1 e ate = data2.

var validou = f.deMenorQueAte('data1', 'data2);

validaCampo(idOuClasse)

Este metodo valida o campo recebendo um id ou classe. Verifica se o campo não esta em branco, se ele esta retorna false, senão atribui o campo ao objeto do Formulario de dados e retorna true.

var validou = f.validaCampo('grupos');

validaArquivo(ID, arrayDeTipos)

Este metodo valida se o arquivo ou arquivos enviados no campo file onde o ID foi informado atende aos tipos de arquivos listado no arrayDeTipos. Se todos atendem retorna true, senão exibem uma mensagem dizendo que tem arquivo de tipo incorreto e retorna false.

var validou = f.validaArquivo('arquivo', ['.png', '.PNG', '.JPG', '.jpg', '.JPEG', '.jpeg']);

O objeto dados do Formulario

Quando você utiliza metodos como deMenorQueAte() e validaCampo() eles depois de validados criam atributos no objeto de Formulario chamado dados. Para recuperar os dados que foram validados basta você chamar o atributo dados. Usando nosso exemplo vamos chama-lo em f.

var dados = f.dados

Os dados retornados são um par de chave e valor, sendo que deMenorQueAte cria as chaves de e ate, e validaCampo cria a chave do nome do idOuClasse do campo. f.dados.de;f.dados.ate;

converter(valor)

Converte um valor repassado como string em um monetario com a sigla R$ e o retorna. Caso seja um float pode converte-lo usando o metodo toFixed(2) de parseFloat para envia-lo corretamente.

var valor = 15000.00;var conv = converter(parseFloat(valor).toFixed(2));
