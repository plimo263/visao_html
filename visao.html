Class ClasseId

Descrição: Esta classe tem o objetivo de disponibilizar atributos comuns entre elementos de uma pagina. O atributo class, id, e algum atributo personalizado.

NOTA: ESTA CLASSE É USADA COMO HERANÇA DAS DEMAIS, NÃO TEM UTILIDADE, SOMENTE SEUS METODOS E ATRIBUTOS SAO HERDADOS PARA SIMPLIFICAR A CRIAÇÃO DE CLASSES MAIS ESPECIFICAS.

  • Atributos
  • Métodos

String: classe -> Uma string que representa a classe de qualquer elemento html.

String: id -> Uma string que representa o id de qualquer elemento.


CONSTRUTOR: ClasseId(String classe, String id)

Constrói um objeto que vai conter representações de classe, id e atributos.

setClasse(String classe)

Define o valor do atributo class.

setId(String id)

Define o valor do atributo id.

setConteudo(String conteudo)

Define o conteudo de um elemento html.

addAtributo(String atributo)

Adiciona atributos ao elemento html em questão.

getClasse()

Recupera o valor da classe

getId()

Recupera o valor do Id

getConteuto()

Recupera o valor do conteudo

getAtributo()

Recupera o valor dos atributos.

Class Para

Descrição: Esta classe tem objetivo de gerar uma representação do elemento p.

A classe Para herda de ClasseId.

  • Atributos
  • Métodos

String: conteudo -> Conteudo que sera anexado entre o elemento p

String: classe -> Nomes de classes para ser usado pelo elemento p

String: id -> Nome para ser usado como id do elemento p.


CONSTRUTOR: Para(String conteudo, String classe, String id)

Constrói o objeto da Classe Para enviando o conteudo a classe e o id do elemento p.

Exemplo:

var p = new Para('Ola', 'text-center, 'paragrafo1')

setPara(String conteudo)

Recebe uma string que será o conteúdo dentro da tag p.

p.setPara('OLA MUNDO');

getPara()

Retorna uma representação do elemento p

p.getPara();

Class Tabela

Descrição: Esta classe cria objetos que conseguem representar um elemento de tabela html.

A classe Tabela herda de ClasseId.

  • Atributos
  • Métodos

Array: cabecalho -> Este define o cabecalho de uma tabela html (thead). Um array simples.

Array[Array] : corpo -> Este define o corpo da tabela html (tbody). Varios arrays dentro de um array, cada array será como uma linha da tabela. NOTA: O TAMANHO (array.length) DE CADA ARRAY INTERNO DEVE SER O MESMO DO CABECALHO.

String: classe -> Uma classe para a tabela html.

String: id -> Um id para a tabela html.

String: classeCabecalho -> A classe do cabecalho (thead) da tabela.


CONSTRUTOR: Tabela(Array cabecalho, Array[Array] corpo, String classe, String id, String classeCabecalho)

Instancia um objeto Tabela com seu cabecalho e corpo.

var t = new Tabela(['NOME','SOBRENOME'],[['MARCOS', 'FELIPE'],['WILSON', 'JARDIM']], 'table table-hover', '', '')

setClasseCabecalho(String classe)

Define a classe do cabecalho da tabela html.

t.setClasseCabecalho('bg-success');

setCorpo(Array[Array] corpo)

Define o corpo de uma tabela html. Ele deve ter a mesma quantidade de elementos em cada array interno e cada array interno deve ter o tamanho igual o do cabecalho.

t.setCorpo([['JOAO', 'BARBOSA']]);

setRodape(Array rodape)

Define o rodape da tabela, este por padrão não é definido no construtor. É esperado um array com o mesmo tamanho (quantidade de itens) do cabecalho.

t.setRodape(['NOME','SOBRENOME'])

converterMonetario(int indexColuna)

Metodo utilizado para converter uma determinada coluna em monetario, por exemplo, se você quiser converter a primeira coluna use o indice 0.

t.converterMonetario(0); // Converter a primeira coluna

converterPercentual(int indexColuna)

Metodo usado para converter uma determinada coluna em monetario, por exemplo, se você quiser converter a primeira coluna, use o indice 0

t.converterPercentual(0); // Converte a primeira coluna em percentual

desenhaDataTable()

Este metodo funciona com o apoio da biblioteca dataTables, ele desenha uma tabela usando a api do dataTables (claro que a tabela já deve existir no html, veja getTabela())

t.desenhaDataTable();

getTabelaDinamica(int colunaChave, Array valoresMonetarios, int campoAserMovido)

Um dos métodos mais complexos utilizados pelo objeto Tabela. Ele tem 3 campos chaves para configuração, cada um deles determina a forma como a tabela dinamica sera desenhada. O primeiro é a colunaChave, ele vai centralizar registros cujo esta coluna seja identica a outras, por exemplo, se o corpo tem esta aparencia, [['MARCOS', 320], ['FELIPE', 230], ['MARCOS', 420]] isto que dizer que a colunaChave zero vai unificar todos os nomes iguais na coluna, se for a coluna zero, então teremos somente um registro de MARCOS e um de FELIPE.

O segundo parametro define colunas como valores monetarios, ou seja, valores que serão somados para definir um total a cada coluna e automaticamente formatados, como o corpo do exemplo acima a coluna 1 pode ser informada dentro de um array ([1]) como coluna que contem valores monetarios a serem totalizados e convertidos.

O terceiro e ultimo parametro, não o menos importante, move um determinado campo para a posição do campo da colunaChave, onde os valores duplicados são removidos. Não compreendeu ? Vou explicar. Vamos supor que temos o seguinte corpo. [['MARCOS','CC',280],['FELIPE','CC',480],['MARCOS','R$',110]]. Foi escolhida como coluna a ser movida a coluna 1, e a coluna a ser convertida a coluna 2, então nos registros MARCOS CC e R$ serão movidos abaixo do nome, assim como FELIPE recebera abaixo o CC, os valores serao somados e o resultado vai dar uma ideia de tabela dinamina sendo que a representacao sera exibida desta forma [['MARCOS','--',R$ 390],['MARCOS','R$',R$ 110],['MARCOS','CC',R$ 280],['FELIPE','--',R$ 480],['FELIPE','CC',R$ 480]]

Um exemplo de execucao do codigo

t.getTabelaDinamica(0,[2],1)

clickTabelaDinamica()

Metodo usado para definir o evento click a tabela dinamica já criada. Este metodo é totalmente dependente de getTabelaDinamica(). Para funcionar é necessário que a tabela dinamica já esteja desenhada na pagina.

t.clickTabelaDinamica()

getTabela()

Recupera uma representação html de uma tabela, com seu cabecalho, corpo e (se houver) seu rodape, suas classes e id. Caso o corpo não esteja de acordo com o tamanho do cabecalho uma mensagem de erro será retornada.

t.getTabela();

removeColuna(int indexColuna)

Remove a coluna da tabela recebendo seu indice, o valor do indice inicia-se por 0, sendo que a primeira coluna tem indice 0.

t.removeColuna(1);

filtro(String palavraChave)

Metodo usado para retornar cabecalho e corpo de dados filtrados. Recebe uma string para filtro e retorna [[cabecalho],[[corpo],[corpo]]]

var recebeNovaTabela = t.filtro('Marcos');

Class Lista

Descrição: Cria um objeto capaz de representar uma lista, seja ela ordenada ou não.

A classe Lista herda de ClasseId.

  • Atributos
  • Métodos

Array: itens -> Um array com os itens que faram parte da lista (cada li).

String: classe -> Uma classe para fazer parte da lista.

String: id -> Um id para fazer parte da lista.


CONSTRUTOR: Lista(Array itens, boolean ordenada, String classe, String id)

Instancia um objeto Lista para um elemento de lista html.

var l = new Lista(['ovo','manteiga'], true, '','');

getLista()

Retorna uma lista ordenada ou não dependendo do parametro que você enviou.

l.getLista();

Class Selecao

Descrição: Cria um objeto Selecao capaz de representar um elemento select.

A classe Selecao herda de ClasseId.

  • Atributos
  • Métodos

String: nome -> Um nome para o select, este é o valor do atributo name

String: classe -> Define uma classe para o elemento select.

String: id -> Define um id para o elemento select.


CONSTRUTOR Selecao(String nome, String classe, String id)

Instancia um objeto Selecao que sera capaz de representar uma tag select.

var selecao = new Selecao('nome','','');

addItem(Array item)

Apenda um item ao select, resumindo cria mais um option para o select. O array enviado deve ter dois elementos, um sendo o value e o outro o rotulo.

selecao.setItem(['1','UM']);

addItens(Array[Array] item)

Apenda todos os itens do array ao select. Deve ser um array com outro array alinhado e o tamanho de cada array alinhado deve ser 2.

selecao.setItens([['2','DOIS'],['3','TRES']]);

getSelecao()

Retorna a representação de uma tag select com todos o itens (option) disponveis.

selecao.getSelecao();

Class DivRow

Descrição: Classe usada para instanciar um objeto DivRow que vai representar uma divRow do bootstrap

A classe DivRow herda de ClasseId.

  • Atributos
  • Métodos

String: classe -> Define a classe de um objeto divRow

String: id -> Define o id para o objeto divRow

Integer: tamanho -> Define a quantidade de divs col-ms vão existir, somente é permitida 12 destas.


CONSTRUTOR: DivRow(String classe, String id)

Este metodo contrói um objeto DivRow capaz de representar divs de 12 grades no máximo, o tamanho é definido na chamada do método addDiv().

var di = new DivRow();

addDiv(String conteudo, Integer tamanho, String classe, String id);

Este método é a melhor forma de conversar com um objeto DivRow, com este metodo pode-se informar o conteudo, o tamanho, sendo que a soma dos tamanhos deve ser igual ou inferior a 12, a classe da div e o id da div que será adicionada.

di.addDiv('oi',4,'','');

getDivRow()

Retorna um elemento div com divs alinhadas.

di.getDivRow();

Class DivTabs

Descrição: Classe usada para instanciar um objeto DivTabs que vai representar uma divTabs do bootstrap

A classe DivTabs herda de ClasseId.

  • Atributos
  • Métodos

Array: lista -> Uma lista que vai definir os links da tabulacao.

Array: conteudo -> Um array com o conteudo de cada aba do divTabs.

ESTES ATRIBUTOS NÃO SÃO DEFINIDOS PELO USUÁRIO, O PREENCHIMENTO DELES SE DÁ PELO METODO addDivTabs que é demonstrado na aba dos métodos.

CONSTRUTOR: DivTabs()

Contrói um objeto divTabs capaz de representar a tabela.

var diT = new DivTabs();

addDivTabs(String nome, String idDaDiv, String Conteudo

O nome é de fato o que aparece na aba, já o idDaDiv é o id que vai ligar a aba ate a div contida no divTabs, já o conteudo é de fato o que vai dentro da aba.

diT.addDivTabs('NOMES', 'nomes', '<p>Marcos</p>')

getDivTabs()

Retorna a representaçao da divTabs para ser inserida na pagina como voce desejar.

diT.getDivTabs();

Class Modal

Descrição: Classe usada para instanciar um objeto Modal que vai representar um modal do bootstrap

A classe Modal herda de ClasseId.

  • Atributos
  • Métodos

String: cabeModal -> Conteudo do cabecalho do modal

String: corpoModal -> Conteudo do corpo do modal

String: rodapeModal -> Conteudo do rodape do modal

boolean: tipoModal -> Conteudo do tipo do modal


CONSTRUTOR Modal(String cabeModal, String corpoModal, String rodapeModal, String classe, String id)

Este é o construtor do modal, recebe estes parametos posicionais para se criar um modal com largura media.

var mo = new Modal('<h1>Ola</>', '<p>TESTE</p>', '', '', '')

setCabeModal(String cabeModal);

Recebe uma string que vai representar o cabecalho do modal.

mo.setCabeModal('');

setCorpoModal(String corpoModal);

Define o corpo do modal recebendo a string que vai representa-lo.

mo.setCorpoModal('');

setRodapeModal(String rodaModal)

Define o rodape do modal recebendo uma string que vai representa-lo.

mo.setRodapeModal('');

setTipoModal(boolean tipoModal)

Recebe um booleando definindo o tipo do modal. Se tipoModal for true o modal fica com fundo preto, mas clicando fora ele fecha, se for false ele fica com fundo transparente e não é possivel fecha-lo clicando fora e se for enviado a string 'static' o modal fica com fundo preto e não é fechado clicando fora, somente fechando pelos seus botoes.

mo.setTipoModal(true);

getModal()

Obtem a representacao html do modal, este pode ser usado para ser incluso no documento html.

var modal = mo.getModal();

executaModal()

Este metodo executa o modal no estilo javascript sendo que se o modal já estiver contido no documento html ele o faz aparecer na tela.

mo.executaModal();
